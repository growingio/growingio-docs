# 广告相关数据

## ads\_track\_activation——广告激活事件

激活事件指GrowingIO 移动端Android/iOS sdk采集上报的激活事件数据。

 如果激活事件能归因到某次点击事件，其中linkId,campaignId,channelId,linkName,campaignName,channelName,adsVariable 是对应点击事件的链接信息。
