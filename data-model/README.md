# 数据模型

## 数据模型简介

GrowingIO做的是用户行为数据分析产品，顾名思义，核心问题是：用户、行为、分析

为了通过数据来客观准确的展现这三个核心概念，GrowingIO根据多年用户行为分析的经验，建立了一整套标准化的用户模型、事件模型和统计模型。

## 数据模型的相互关系

* 事件模型基于用户模型
* 分析模型基于事件模型
